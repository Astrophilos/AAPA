# AAPA
This is the GitHub repository for AAPA (Astrophilos Automated Polar Alignment).

All the firmware and software are fully open source.
The CAD files are not included in this repository, and can be bought at: https://astrophiloslab.com/aapa

We currently have 3 main files:

1) AAPA_for_software.ino  -> Firmware for the FYSETC E4 board (ESP32)
2) Platedual.py -> Python source code used to communicate with AAPA and read NINA TPPA log files
3) Platedual.exe  -> Precompiled executable version of the software, ready to use

The full project is documented step-by-step here:
